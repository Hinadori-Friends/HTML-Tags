# HTMLタグ解説サイト

WHATWG HTML Living Standard の Indices（Elements）に準拠した、HTMLタグの学習用サイトです。全タグの解説ページを用意し、初心者が理解しやすい説明とコード例を掲載します。Obsolete features に掲載されている廃止・非推奨要素は対象外です。

## 構成
- `index.html`: タグ一覧と各タグページへのリンク
- `common-attributes.html`: 共通属性の一覧ページ
- `style.css`: 共通スタイル
- `tag/`: 各タグの解説ページ（`<タグ名>.html`）
- `spec/plan.md`: 計画・進行状況
- `spec/gpt.md`: 仕様（GPT向け）
- `spec/chat.md`: チャットログ

## 使い方
1. `index.html` をブラウザで開く
2. 調べたいタグのページへ移動する
3. 共通属性は `common-attributes.html` を参照する

## 補足
- 固有属性は折りたたみ表示で整理しています。
- 選択肢がある属性は各値の意味も説明しています。
- 外部ファイルを参照できるタグには読み込み方法も記載しています。
