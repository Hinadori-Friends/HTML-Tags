# HTMLタグ解説サイト

WHATWG HTML Living Standard の Indices（Elements）を基準に、HTMLタグの一覧と個別解説ページをまとめた学習用サイトです（Obsolete features の要素は対象外）。

## 構成
- `index.html`: タグ一覧と各ページへのリンク
- `tag/`: 各タグの解説ページ（`<タグ名>.html`）
- `spec/plan.md`: 計画
- `spec/gpt.md`: 仕様

## 使い方
1. `index.html` をブラウザで開きます。
2. 一覧からタグを選んで解説ページに移動します。

## 備考
- 各タグページには簡単な説明とコード例（コードブロック）を掲載しています。
- 一部の要素は構造・メタデータ用途のため、実行例はコードのみを表示しています。
