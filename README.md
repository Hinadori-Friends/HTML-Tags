# HTMLタグ解説サイト

WHATWG HTML Living Standard の Indices（Elements）を基準に、HTMLタグの一覧と個別解説ページをまとめた学習用サイトです（Obsolete features の要素は対象外）。

## 構成
- `index.html`: タグ一覧と各ページへのリンク
- `common-attributes.html`: 共通属性の一覧
- `style.css`: 共通スタイル
- `tag/`: 各タグの解説ページ（`<タグ名>.html`）
- `spec/plan.md`: 計画
- `spec/gpt.md`: 仕様
- `spec/chat.md`: チャットログ

## 使い方
1. `index.html` をブラウザで開きます。
2. 一覧からタグを選んで解説ページに移動します。
3. 共通属性は `common-attributes.html` で確認できます。

## 備考
- 各タグページには説明・コード例・属性一覧（固有属性は折りたたみ）を掲載しています。
- 外部ファイルを参照できるタグには、読み込み方法の説明も追加しています。
