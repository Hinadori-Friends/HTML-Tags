<a> タグ

HTMLの <a> 要素は、Hyperlink を表します。 リンク先を示す href は必須です。 リンク文は「こちら」ではなく内容が分かる言葉にします。 同じページ内リンクならIDを使います。 関連タグ: <link>、<nav>。

属性

固有属性（クリックで開く）
href
意味: リンク先URLを指定します。
指定方法: href="値"
target
意味: リンク先の開き方を指定します。
指定方法: target="値"
選択肢:
  • _self: 同じタブ（既定）で開く
  • _blank: 新しいタブ/ウィンドウで開く
  • _parent: 親フレームで開く
  • _top: 最上位の文書で開く
download
意味: リンク先をダウンロードとして扱います。
指定方法: download="値"
ping
意味: URLを指定します。
指定方法: ping="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
rel
意味: リンク先との関係を指定します。
指定方法: rel="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
hreflang
意味: リンク先の言語を指定します。
指定方法: hreflang="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"
referrerpolicy
意味: リファラ送信方針を指定します。
指定方法: referrerpolicy="値"
選択肢:
  • no-referrer: リファラを送らない
  • no-referrer-when-downgrade: HTTPS→HTTPでは送らない
  • origin: オリジンのみ送る
  • origin-when-cross-origin: 同一オリジンは完全、他はオリジンのみ
  • same-origin: 同一オリジンのみ送る
  • strict-origin: オリジンのみ、ダウングレード時は送らない
  • strict-origin-when-cross-origin: 同一は完全、他はオリジンのみ、ダウングレード時は送らない
  • unsafe-url: 完全なURLを常に送る

コード例

<a href="https://example.com">例へのリンク</a>
