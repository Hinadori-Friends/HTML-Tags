<a> タグ

HTMLの <a> 要素は、Hyperlink を表します。 リンク先を示す href は必須です。 リンク文は「こちら」ではなく内容が分かる言葉にします。 同じページ内リンクならIDを使います。 関連タグ: <link>、<nav>。

属性

固有属性（クリックで開く）
href

何を表すか: リンク先を指定する属性です。クリックするとこのURLへ移動します。相対パスも使えます。

ポイント: リンク先を決める一番大切な属性です。まずは有効なURLか相対パスを書くことを意識します。

書き方: href="値"

例: <a href="例">...</a>

target

何を表すか: リンクをどこで開くかを指定します。新しいタブにするか同じタブにするかを決めます。

ポイント: リンクの開き方を決めます。新しいタブにしたい場合によく使います。

書き方: target="値"

選択肢:

  • _self: 同じタブ（既定）で開く
  • _blank: 新しいタブ/ウィンドウで開く
  • _parent: 親フレームで開く
  • _top: 最上位の文書で開く

例: <a target="例">...</a>

download

何を表すか: リンク先を「ダウンロード」として扱う指示です。クリック時に保存ダイアログが出ます。

ポイント: 指定するとリンク先をダウンロードとして扱います。

書き方: download="値"

例: <a download="例">...</a>

ping

何を表すか: URLを指定します。例: 読み込むファイルや移動先のURLを入れます。

書き方: ping="値"

選択肢:

  • スペース区切り: 複数キーワードを指定

例: <a ping="例">...</a>

rel

何を表すか: リンク先との関係を指定します。スタイルシート・外部参照・nofollow など用途を伝えます。

ポイント: リンクの関係を示します。外部リンクやスタイル指定などで使います。

書き方: rel="値"

選択肢:

  • スペース区切り: 複数キーワードを指定

例: <a rel="例">...</a>

hreflang

何を表すか: リンク先の言語を指定します。例: 英語ページなら en、日本語なら ja を指定します。

書き方: hreflang="値"

例: <a hreflang="例">...</a>

type

何を表すか: 要素の種類やデータの形式を指定します。指定するとブラウザが正しく解釈しやすくなります。

ポイント: 要素の種類や形式を決めます。用途が複数ある要素では特に重要です。

書き方: type="値"

例: <a type="例">...</a>

referrerpolicy

何を表すか: 外部スクリプト取得時に送るリファラ（参照元情報）の方針を指定します。

書き方: referrerpolicy="値"

選択肢:

  • no-referrer: リファラを送らない
  • no-referrer-when-downgrade: HTTPS→HTTPでは送らない
  • origin: オリジンのみ送る
  • origin-when-cross-origin: 同一オリジンは完全、他はオリジンのみ
  • same-origin: 同一オリジンのみ送る
  • strict-origin: オリジンのみ、ダウングレード時は送らない
  • strict-origin-when-cross-origin: 同一は完全、他はオリジンのみ、ダウングレード時は送らない
  • unsafe-url: 完全なURLを常に送る

例: <a referrerpolicy="例">...</a>

コード例

<a href="https://example.com">例へのリンク</a>
