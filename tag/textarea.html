<textarea> タグ

HTMLの <textarea> 要素は、Multiline text controls を表します。 複数行の入力欄を作ります。 行数や列数は rows と cols で指定します。 長文の入力に向いています。 関連タグ: <input>、<label>。

属性

固有属性（クリックで開く）
autocomplete

何を表すか: 入力補完の方針を指定します。例: email や ame を指定して、ブラウザに入力の意味を伝えます。

書き方: autocomplete="値"

選択肢:

  • on: 補完を許可
  • off: 補完を無効化
  • 用途別トークン: 例: name, email, username, current-password など

例: <textarea autocomplete="例">...</textarea>

cols

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: cols="値"

例: <textarea cols="例">...</textarea>

dirname

何を表すか: 入力方向を送信データに含めます。指定すると name.dir の形で方向情報が送信されます。

書き方: dirname="値"

例: <textarea dirname="例">...</textarea>

disabled

何を表すか: 操作できない状態にします。送信対象にもならない点に注意します。

ポイント: 操作できない状態にします。見た目も変わることが多いです。

書き方: 属性名だけ書く（例: disabled）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <textarea disabled>...</textarea>

form

何を表すか: この要素がどのformに属するかを指定します。フォームの外に置いた部品を、特定のフォームに結び付けたいときに使います。

書き方: form="値"

例: <textarea form="例">...</textarea>

maxlength

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: maxlength="値"

例: <textarea maxlength="例">...</textarea>

minlength

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: minlength="値"

例: <textarea minlength="例">...</textarea>

name

何を表すか: フォーム送信時の「項目名」になる属性です。サーバー側ではこの名前で値を受け取ります。

ポイント: 送信時のキーになる名前です。フォームでは必須になることが多いです。

書き方: name="値"

例: <textarea name="例">...</textarea>

placeholder

何を表すか: 入力欄に薄いヒント文字を表示します。入力すると消えます。

書き方: placeholder="値"

例: <textarea placeholder="例">...</textarea>

readonly

何を表すか: 読み取り専用にします。表示はされますが編集できません。

ポイント: 表示はするが編集できない状態にします。

書き方: 属性名だけ書く（例: readonly）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <textarea readonly>...</textarea>

required

何を表すか: 必須入力であることを示します。空欄だと送信できなくなります。

ポイント: 必須入力にする属性です。

書き方: 属性名だけ書く（例: required）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <textarea required>...</textarea>

rows

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: rows="値"

例: <textarea rows="例">...</textarea>

wrap

何を表すか: 送信時の改行の扱いを指定します。例: soft（表示のみ改行）/hard（送信値に改行）。

書き方: wrap="値"

例: <textarea wrap="例">...</textarea>

コード例

<label>メッセージ<br><textarea rows="3" cols="20"></textarea></label>
