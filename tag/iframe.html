<iframe> タグ

HTMLの <iframe> 要素は、Child navigable を表します。 画像・動画・外部コンテンツなどを埋め込むときに使います。 代替テキストやサイズ指定を入れると表示とアクセシビリティが向上します。 必要に応じて複数のソースを用意します。 関連タグ: <img>、<video>、<picture> など。

属性

固有属性（クリックで開く）
src
意味: リソースのURLを指定します。
指定方法: src="値"
srcdoc
意味: iframeに埋め込むHTML文字列を指定します。
指定方法: srcdoc="値"
name
意味: 要素の名前（送信時のキーなど）を指定します。
指定方法: name="値"
sandbox
意味: キーワード（複数指定可）を指定します。
指定方法: sandbox="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
allow
意味: 用途に応じた値を指定します。
指定方法: allow="値"
allowfullscreen
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: allowfullscreen）
選択肢:
  • 付与: 有効
  • 省略: 無効
width
意味: 表示幅を指定します。
指定方法: width="値"
height
意味: 表示高さを指定します。
指定方法: height="値"
referrerpolicy
意味: リファラ送信方針を指定します。
指定方法: referrerpolicy="値"
選択肢:
  • no-referrer: リファラを送らない
  • no-referrer-when-downgrade: HTTPS→HTTPでは送らない
  • origin: オリジンのみ送る
  • origin-when-cross-origin: 同一オリジンは完全、他はオリジンのみ
  • same-origin: 同一オリジンのみ送る
  • strict-origin: オリジンのみ、ダウングレード時は送らない
  • strict-origin-when-cross-origin: 同一は完全、他はオリジンのみ、ダウングレード時は送らない
  • unsafe-url: 完全なURLを常に送る
loading
意味: 遅延読み込みの方針を指定します。
指定方法: loading="値"
選択肢:
  • lazy: 画面に近づいたら読み込む
  • eager: すぐに読み込む

コード例

<iframe srcdoc="<p>埋め込みコンテンツ</p>" title="例"></iframe>
