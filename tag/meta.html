<meta> タグ

HTMLの <meta> 要素は、Text metadata を表します。 文字コードや表示設定など、ページの基本情報を指定します。 この要素は空要素で終了タグを持ちません。 設定を間違えると表示が崩れることがあります。 関連タグ: <head>、<link>。

属性

固有属性（クリックで開く）
name
意味: 要素の名前（送信時のキーなど）を指定します。
指定方法: name="値"
http-equiv
意味: HTTPヘッダー相当の指示を指定します。
指定方法: http-equiv="値"
content
意味: meta要素の内容を指定します。
指定方法: content="値"
charset
意味: 文字コードを指定します。
指定方法: charset="値"
media
意味: 用途に応じた値を指定します。
指定方法: media="値"

コード例

<!doctype html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>
<body>...</body>
</html>

実行例

この要素は文書構造・メタデータ・テンプレート用途のため、実行例はコードのみ示します。
