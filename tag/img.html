<img> タグ

HTMLの <img> 要素は、Image を表します。 画像を表示する要素で、src と alt を必ず指定します。 この要素は空要素で終了タグを持ちません。 サイズを指定するとレイアウトが安定します。 関連タグ: <picture>、<source>。

属性

共通属性

id
意味: ページ内で唯一の識別子を付けます。 指定方法: id="main"
選択肢: ユニークな文字列
class
意味: CSSやJSで使うクラス名を付けます。 指定方法: class="card"
選択肢: 複数指定可（スペース区切り）
title
意味: 補足説明（ツールチップ）を付けます。 指定方法: title="説明"
選択肢: テキスト
lang
意味: 要素の言語を指定します。 指定方法: lang="ja"
選択肢: BCP 47 言語タグ
dir
意味: 文字の方向を指定します。 指定方法: dir="ltr"
選択肢: ltr / rtl / auto
hidden
意味: 要素を非表示にします。 指定方法: hidden
選択肢: 付与で非表示
tabindex
意味: Tab移動の順序やフォーカス可否を指定します。 指定方法: tabindex="0"
選択肢: 整数
style
意味: 要素に直接CSSを書くときに使います。 指定方法: style="color: red;"
選択肢: CSS宣言
contenteditable
意味: 要素内を編集可能にします。 指定方法: contenteditable="true"
選択肢: true / false
spellcheck
意味: スペルチェックの有無を指定します。 指定方法: spellcheck="true"
選択肢: true / false
translate
意味: 翻訳の対象にするか指定します。 指定方法: translate="no"
選択肢: yes / no
draggable
意味: ドラッグ可能か指定します。 指定方法: draggable="true"
選択肢: true / false
inert
意味: 操作やフォーカスを受け付けない状態にします。 指定方法: inert
選択肢: 付与で無効化
data-*
意味: 独自データ属性を定義します。 指定方法: data-user-id="123"
選択肢: 任意の文字列
aria-*
意味: アクセシビリティ用の属性です。 指定方法: aria-label="説明"
選択肢: 属性ごとに規定
role
意味: ARIAの役割を指定します。 指定方法: role="button"
選択肢: ARIAロール

固有属性

alt
意味: 画像や領域の代替テキストを指定します。
指定方法: alt="値"
src
意味: リソースのURLを指定します。
指定方法: src="値"
srcset
意味: レスポンシブ画像の候補を指定します。
指定方法: srcset="値"
選択肢: 複数キーワード（カンマ区切り）
sizes
意味: レスポンシブ画像の表示幅条件を指定します。
指定方法: sizes="値"
選択肢: 複数キーワード（スペース区切り）
crossorigin
意味: CORSの送信方式を指定します。
指定方法: crossorigin="値"
選択肢: anonymous / use-credentials
usemap
意味: 用途に応じた値を指定します。
指定方法: usemap="値"
ismap
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: ismap）
選択肢: 付与で有効 / 省略で無効
width
意味: 表示幅を指定します。
指定方法: width="値"
height
意味: 表示高さを指定します。
指定方法: height="値"
referrerpolicy
意味: リファラ送信方針を指定します。
指定方法: referrerpolicy="値"
選択肢: no-referrer / no-referrer-when-downgrade / origin / origin-when-cross-origin / same-origin / strict-origin / strict-origin-when-cross-origin / unsafe-url
decoding
意味: 画像デコードの方針を指定します。
指定方法: decoding="値"
選択肢: auto / sync / async
loading
意味: 遅延読み込みの方針を指定します。
指定方法: loading="値"
選択肢: lazy / eager
fetchpriority
意味: 用途に応じた値を指定します。
指定方法: fetchpriority="値"
