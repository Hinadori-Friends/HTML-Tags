<ol> タグ

HTMLの <ol> 要素は、Ordered list を表します。 順序付きの一覧を作ります。 各項目は <li> で書きます。 順序が不要なら <ul> を使います。 関連タグ: <li>、<ul>。

属性

固有属性（クリックで開く）
reversed

何を表すか: 指定の有無で有効/無効を切り替えます。指定するとON、外すとOFFになります。

書き方: 属性名だけ書く（例: reversed）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <ol reversed>...</ol>

start

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: start="値"

例: <ol start="例">...</ol>

type

何を表すか: 要素の種類やデータの形式を指定します。指定するとブラウザが正しく解釈しやすくなります。

ポイント: 要素の種類や形式を決めます。用途が複数ある要素では特に重要です。

書き方: type="値"

例: <ol type="例">...</ol>

コード例

<ol>
  <li>手順1</li>
  <li>手順2</li>
</ol>

実行例

  1. 手順1
  2. 手順2
