<ol> タグ

HTMLの <ol> 要素は、Ordered list を表します。 順序付きの一覧を作ります。 各項目は <li> で書きます。 順序が不要なら <ul> を使います。 関連タグ: <li>、<ul>。

属性

固有属性（クリックで開く）
reversed
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: reversed）
選択肢:
  • 付与: 有効
  • 省略: 無効
start
意味: 数値を指定します。
指定方法: start="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"

コード例

<ol>
  <li>手順1</li>
  <li>手順2</li>
</ol>

実行例

  1. 手順1
  2. 手順2
