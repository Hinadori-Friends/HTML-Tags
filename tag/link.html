<link> タグ

HTMLの <link> 要素は、Link metadata を表します。 外部のCSSなど、関連リソースを読み込むために使います。 この要素は空要素で終了タグを持ちません。 主に <head> 内で使います。 関連タグ: <style>、<meta>。

外部CSSを読み込む代表例は rel="stylesheet" と href です。例: <link rel="stylesheet" href="style.css">。

属性

固有属性（クリックで開く）
href
意味: リンク先URLを指定します。
指定方法: href="値"
crossorigin
意味: CORSの送信方式を指定します。
指定方法: crossorigin="値"
選択肢:
  • anonymous: 資格情報（Cookie等）を送らない
  • use-credentials: 資格情報を送る
rel
意味: リンク先との関係を指定します。
指定方法: rel="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
as
意味: 用途に応じた値を指定します。
指定方法: as="値"
media
意味: 用途に応じた値を指定します。
指定方法: media="値"
hreflang
意味: リンク先の言語を指定します。
指定方法: hreflang="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"
sizes
意味: レスポンシブ画像の表示幅条件を指定します。
指定方法: sizes="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
imagesrcset
意味: 用途に応じた値を指定します。
指定方法: imagesrcset="値"
選択肢:
  • カンマ区切り: 複数キーワードを指定
imagesizes
意味: 用途に応じた値を指定します。
指定方法: imagesizes="値"
referrerpolicy
意味: リファラ送信方針を指定します。
指定方法: referrerpolicy="値"
選択肢:
  • no-referrer: リファラを送らない
  • no-referrer-when-downgrade: HTTPS→HTTPでは送らない
  • origin: オリジンのみ送る
  • origin-when-cross-origin: 同一オリジンは完全、他はオリジンのみ
  • same-origin: 同一オリジンのみ送る
  • strict-origin: オリジンのみ、ダウングレード時は送らない
  • strict-origin-when-cross-origin: 同一は完全、他はオリジンのみ、ダウングレード時は送らない
  • unsafe-url: 完全なURLを常に送る
integrity
意味: 任意のテキストを指定します。
指定方法: integrity="値"
blocking
意味: キーワード（複数指定可）を指定します。
指定方法: blocking="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
color
意味: 用途に応じた値を指定します。
指定方法: color="値"
disabled
意味: 操作できない状態にします。
指定方法: 属性名だけ書く（例: disabled）
選択肢:
  • 付与: 有効
  • 省略: 無効
fetchpriority
意味: 用途に応じた値を指定します。
指定方法: fetchpriority="値"

コード例

<!doctype html>
<html lang="ja">
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>...</body>
</html>
