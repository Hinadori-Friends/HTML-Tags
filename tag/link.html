<link> タグ

HTMLの <link> 要素は、Link metadata を表します。 外部のCSSなど、関連リソースを読み込むために使います。 この要素は空要素で終了タグを持ちません。 主に <head> 内で使います。 関連タグ: <style>、<meta>。

属性

共通属性

id
意味: ページ内で唯一の識別子を付けます。 指定方法: id="main"
選択肢:
  • 任意の文字列: 同じページ内で重複しない値にします。
class
意味: CSSやJSで使うクラス名を付けます。 指定方法: class="card"
選択肢:
  • 複数指定: スペース区切りで複数付けられます。
title
意味: 補足説明（ツールチップ）を付けます。 指定方法: title="説明"
選択肢:
  • 任意の文字列: 短い説明文を入れます。
lang
意味: 要素の言語を指定します。 指定方法: lang="ja"
選択肢:
  • BCP 47: 例: ja, en, en-US など。
dir
意味: 文字の方向を指定します。 指定方法: dir="ltr"
選択肢:
  • ltr: 左から右（既定）
  • rtl: 右から左
  • auto: 内容から自動判定
hidden
意味: 要素を非表示にします。 指定方法: hidden
選択肢:
  • 付与: 表示・操作の対象外になります。
tabindex
意味: Tab移動の順序やフォーカス可否を指定します。 指定方法: tabindex="0"
選択肢:
  • 整数: 0は通常、-1はTab移動から除外。
style
意味: 要素に直接CSSを書くときに使います。 指定方法: style="color: red;"
選択肢:
  • CSS宣言: プロパティ: 値; の形式。
contenteditable
意味: 要素内を編集可能にします。 指定方法: contenteditable="true"
選択肢:
  • true: 編集可能にする
  • false: 編集不可にする
spellcheck
意味: スペルチェックの有無を指定します。 指定方法: spellcheck="true"
選択肢:
  • true: 有効にする
  • false: 無効にする
translate
意味: 翻訳の対象にするか指定します。 指定方法: translate="no"
選択肢:
  • yes: 翻訳対象にする
  • no: 翻訳対象外にする
draggable
意味: ドラッグ可能か指定します。 指定方法: draggable="true"
選択肢:
  • true: ドラッグ可能
  • false: ドラッグ不可
inert
意味: 操作やフォーカスを受け付けない状態にします。 指定方法: inert
選択肢:
  • 付与: 操作やフォーカスを無効化。
data-*
意味: 独自データ属性を定義します。 指定方法: data-user-id="123"
選択肢:
  • 任意の文字列: 用途に合わせた値。
aria-*
意味: アクセシビリティ用の属性です。 指定方法: aria-label="説明"
選択肢:
  • 属性ごと: 各ARIA属性の規定に従います。
role
意味: ARIAの役割を指定します。 指定方法: role="button"
選択肢:
  • ARIAロール: 例: button, dialog など。
固有属性（クリックで開く）
href
意味: リンク先URLを指定します。
指定方法: href="値"
crossorigin
意味: CORSの送信方式を指定します。
指定方法: crossorigin="値"
選択肢:
  • anonymous: 資格情報（Cookie等）を送らない
  • use-credentials: 資格情報を送る
rel
意味: リンク先との関係を指定します。
指定方法: rel="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
as
意味: 用途に応じた値を指定します。
指定方法: as="値"
media
意味: 用途に応じた値を指定します。
指定方法: media="値"
hreflang
意味: リンク先の言語を指定します。
指定方法: hreflang="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"
sizes
意味: レスポンシブ画像の表示幅条件を指定します。
指定方法: sizes="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
imagesrcset
意味: 用途に応じた値を指定します。
指定方法: imagesrcset="値"
選択肢:
  • カンマ区切り: 複数キーワードを指定
imagesizes
意味: 用途に応じた値を指定します。
指定方法: imagesizes="値"
referrerpolicy
意味: リファラ送信方針を指定します。
指定方法: referrerpolicy="値"
選択肢:
  • no-referrer: リファラを送らない
  • no-referrer-when-downgrade: HTTPS→HTTPでは送らない
  • origin: オリジンのみ送る
  • origin-when-cross-origin: 同一オリジンは完全、他はオリジンのみ
  • same-origin: 同一オリジンのみ送る
  • strict-origin: オリジンのみ、ダウングレード時は送らない
  • strict-origin-when-cross-origin: 同一は完全、他はオリジンのみ、ダウングレード時は送らない
  • unsafe-url: 完全なURLを常に送る
integrity
意味: 任意のテキストを指定します。
指定方法: integrity="値"
blocking
意味: キーワード（複数指定可）を指定します。
指定方法: blocking="値"
選択肢:
  • スペース区切り: 複数キーワードを指定
color
意味: 用途に応じた値を指定します。
指定方法: color="値"
disabled
意味: 操作できない状態にします。
指定方法: 属性名だけ書く（例: disabled）
選択肢:
  • 付与: 有効
  • 省略: 無効
fetchpriority
意味: 用途に応じた値を指定します。
指定方法: fetchpriority="値"

コード例

<!doctype html>
<html lang="ja">
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>...</body>
</html>
