<option> タグ

HTMLの <option> 要素は、Option in a list box or combo box control を表します。 選択肢の1項目を表します。 表示名と値を分けたいときは value を指定します。 既定の選択には selected を使います。 関連タグ: <select>、<optgroup>。

属性

固有属性（クリックで開く）
disabled

何を表すか: 操作できない状態にします。送信対象にもならない点に注意します。

ポイント: 操作できない状態にします。見た目も変わることが多いです。

書き方: 属性名だけ書く（例: disabled）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <option disabled>...</option>

label

何を表すか: 表示用の短い名前を指定します。長い文章の代わりに、短いラベルを見せたいときに使います。

書き方: label="値"

例: <option label="例">...</option>

selected

何を表すか: 初期選択状態にします。ページを開いた時点で選択済みにしたい項目に使います。

書き方: 属性名だけ書く（例: selected）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <option selected>...</option>

value

何を表すか: フォームで送信される値や初期値を指定します。実際に送るデータ内容に直結します。

ポイント: 送信される値や初期値を決めます。何が送られるかを意識して書きます。

書き方: value="値"

例: <option value="例">...</option>

コード例

<select>
  <option>選択肢A</option>
  <option>選択肢B</option>
</select>
