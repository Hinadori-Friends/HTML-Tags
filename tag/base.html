<base> タグ

HTMLの <base> 要素は、Base URL and default target navigable for hyperlinks and forms を表します。 主に <head> の中で使い、ページの設定や外部リソースを指定します。 この要素は空要素で終了タグを持ちません。 設定を間違えるとページ全体の挙動に影響するので注意します。 関連タグ: <head>、<meta> など。

属性

固有属性（クリックで開く）
href

何を表すか: リンク先を指定する属性です。クリックするとこのURLへ移動します。相対パスも使えます。

ポイント: リンク先を決める一番大切な属性です。まずは有効なURLか相対パスを書くことを意識します。

書き方: href="値"

例: <base href="例">

target

何を表すか: リンクをどこで開くかを指定します。新しいタブにするか同じタブにするかを決めます。

ポイント: リンクの開き方を決めます。新しいタブにしたい場合によく使います。

書き方: target="値"

選択肢:

  • _self: 同じタブ（既定）で開く
  • _blank: 新しいタブ/ウィンドウで開く
  • _parent: 親フレームで開く
  • _top: 最上位の文書で開く

例: <base target="例">

コード例

<!doctype html>
<html lang="ja">
<head>
  <base href="https://example.com/" target="_blank">
</head>
<body>...</body>
</html>

実行例

この要素は文書構造・メタデータ・テンプレート用途のため、実行例はコードのみ示します。
