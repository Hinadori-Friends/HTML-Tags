<select> タグ

HTMLの <select> 要素は、List box control を表します。 選択肢から1つ（または複数）を選ぶためのリストを作ります。 中に <option> を並べます。 グループ化したい場合は <optgroup> を使います。 関連タグ: <option>、<optgroup>。

属性

固有属性（クリックで開く）
autocomplete

何を表すか: 入力補完の方針を指定します。例: email や ame を指定して、ブラウザに入力の意味を伝えます。

書き方: autocomplete="値"

選択肢:

  • on: 補完を許可
  • off: 補完を無効化
  • 用途別トークン: 例: name, email, username, current-password など

例: <select autocomplete="例">...</select>

disabled

何を表すか: 操作できない状態にします。送信対象にもならない点に注意します。

ポイント: 操作できない状態にします。見た目も変わることが多いです。

書き方: 属性名だけ書く（例: disabled）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <select disabled>...</select>

form

何を表すか: この要素がどのformに属するかを指定します。フォームの外に置いた部品を、特定のフォームに結び付けたいときに使います。

書き方: form="値"

例: <select form="例">...</select>

multiple

何を表すか: 複数選択や複数入力を許可します。複数の項目を同時に選ばせたいときに使います。

書き方: 属性名だけ書く（例: multiple）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <select multiple>...</select>

name

何を表すか: フォーム送信時の「項目名」になる属性です。サーバー側ではこの名前で値を受け取ります。

ポイント: 送信時のキーになる名前です。フォームでは必須になることが多いです。

書き方: name="値"

例: <select name="例">...</select>

required

何を表すか: 必須入力であることを示します。空欄だと送信できなくなります。

ポイント: 必須入力にする属性です。

書き方: 属性名だけ書く（例: required）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <select required>...</select>

size

何を表すか: 数値を指定します。例: 3 と書けば3つ分・3行分など、数量や範囲を数で指定します。

書き方: size="値"

例: <select size="例">...</select>

コード例

<select>
  <option>りんご</option>
  <option>みかん</option>
</select>
