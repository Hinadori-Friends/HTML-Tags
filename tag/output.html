<output> タグ

HTMLの <output> 要素は、Calculated output value を表します。 フォームでユーザーから入力を受け取るときに使います。 送信や検証のために name などの属性を設定することが多いです。 ラベルや説明と組み合わせると操作しやすくなります。 関連タグ: <form>、<input>、<button>。

属性

固有属性（クリックで開く）
for
意味: 用途に応じた値を指定します。
指定方法: for="値"
form
意味: 用途に応じた値を指定します。
指定方法: form="値"
name
意味: 要素の名前（送信時のキーなど）を指定します。
指定方法: name="値"

コード例

<form oninput="result.value = Number(a.value) + Number(b.value)">
  <input id="a" type="number" value="2"> +
  <input id="b" type="number" value="3"> =
  <output name="result">5</output>
</form>

実行例

+ = 5
