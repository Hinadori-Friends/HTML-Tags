<button> タグ

HTMLの <button> 要素は、Button control を表します。 クリック可能なボタンを作ります。 フォーム内では type を指定しないと送信になる点に注意します。 JavaScriptの操作にも使えます。 関連タグ: <form>、<input>。

属性

固有属性（クリックで開く）
command
意味: 用途に応じた値を指定します。
指定方法: command="値"
commandfor
意味: 用途に応じた値を指定します。
指定方法: commandfor="値"
disabled
意味: 操作できない状態にします。
指定方法: 属性名だけ書く（例: disabled）
選択肢:
  • 付与: 有効
  • 省略: 無効
form
意味: 用途に応じた値を指定します。
指定方法: form="値"
formaction
意味: URLを指定します。
指定方法: formaction="値"
formenctype
意味: URLを指定します。
指定方法: formenctype="値"
formmethod
意味: 用途に応じた値を指定します。
指定方法: formmethod="値"
formnovalidate
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: formnovalidate）
選択肢:
  • 付与: 有効
  • 省略: 無効
formtarget
意味: 用途に応じた値を指定します。
指定方法: formtarget="値"
name
意味: 要素の名前（送信時のキーなど）を指定します。
指定方法: name="値"
popovertarget
意味: 用途に応じた値を指定します。
指定方法: popovertarget="値"
popovertargetaction
意味: 用途に応じた値を指定します。
指定方法: popovertargetaction="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"
value
意味: 送信される値や初期値を指定します。
指定方法: value="値"

コード例

<button type="button">送信</button>
