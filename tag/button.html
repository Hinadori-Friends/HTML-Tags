<button> タグ

HTMLの <button> 要素は、Button control を表します。 クリック可能なボタンを作ります。 フォーム内では type を指定しないと送信になる点に注意します。 JavaScriptの操作にも使えます。 関連タグ: <form>、<input>。

属性

固有属性（クリックで開く）
command

何を表すか: ボタンで実行するコマンドの種類を指定します（対応環境限定）。

書き方: command="値"

例: <button command="例">...</button>

commandfor

何を表すか: コマンドの対象となる要素のidを指定します。

書き方: commandfor="値"

例: <button commandfor="例">...</button>

disabled

何を表すか: 操作できない状態にします。送信対象にもならない点に注意します。

ポイント: 操作できない状態にします。見た目も変わることが多いです。

書き方: 属性名だけ書く（例: disabled）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <button disabled>...</button>

form

何を表すか: この要素がどのformに属するかを指定します。フォームの外に置いた部品を、特定のフォームに結び付けたいときに使います。

書き方: form="値"

例: <button form="例">...</button>

formaction

何を表すか: URLを指定します。例: 読み込むファイルや移動先のURLを入れます。

書き方: formaction="値"

例: <button formaction="例">...</button>

formenctype

何を表すか: フォーム送信時のデータのエンコード方式を指定します。例: application/x-www-form-urlencoded / multipart/form-data / text/plain。

書き方: formenctype="値"

例: <button formenctype="例">...</button>

formmethod

何を表すか: 送信時のHTTPメソッドを指定します。例: get や post。

書き方: formmethod="値"

例: <button formmethod="例">...</button>

formnovalidate

何を表すか: 送信時のフォーム検証を無効にします。指定するとバリデーションを飛ばして送信し、省略すると通常どおり検証されます。

書き方: 属性名だけ書く（例: formnovalidate）

選択肢:

  • 付与: 有効
  • 省略: 無効

例: <button formnovalidate>...</button>

formtarget

何を表すか: 送信結果の表示先を指定します。例: _self（同じタブ）/_blank（新しいタブ）。

書き方: formtarget="値"

例: <button formtarget="例">...</button>

name

何を表すか: フォーム送信時の「項目名」になる属性です。サーバー側ではこの名前で値を受け取ります。

ポイント: 送信時のキーになる名前です。フォームでは必須になることが多いです。

書き方: name="値"

例: <button name="例">...</button>

popovertarget

何を表すか: 開閉するpopover要素のidを指定します。

書き方: popovertarget="値"

例: <button popovertarget="例">...</button>

popovertargetaction

何を表すか: popoverの動作を指定します。例: show/hide/toggle。

書き方: popovertargetaction="値"

例: <button popovertargetaction="例">...</button>

type

何を表すか: 要素の種類やデータの形式を指定します。指定するとブラウザが正しく解釈しやすくなります。

ポイント: 要素の種類や形式を決めます。用途が複数ある要素では特に重要です。

書き方: type="値"

例: <button type="例">...</button>

value

何を表すか: フォームで送信される値や初期値を指定します。実際に送るデータ内容に直結します。

ポイント: 送信される値や初期値を決めます。何が送られるかを意識して書きます。

書き方: value="値"

例: <button value="例">...</button>

コード例

<button type="button">送信</button>
