<input> タグ

HTMLの <input> 要素は、Form control を表します。 フォーム入力の中心的な要素で、用途に応じて type を選びます。 この要素は空要素で終了タグを持ちません。 ラベルと関連付けると操作しやすくなります。 関連タグ: <label>、<form>。

属性

固有属性（クリックで開く）
accept
意味: 受け付けるファイル種別を指定します。
指定方法: accept="値"
選択肢:
  • カンマ区切り: 複数キーワードを指定
alpha
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: alpha）
選択肢:
  • 付与: 有効
  • 省略: 無効
alt
意味: 画像や領域の代替テキストを指定します。
指定方法: alt="値"
autocomplete
意味: 入力補完の方針を指定します。
指定方法: autocomplete="値"
選択肢:
  • on: 補完を許可
  • off: 補完を無効化
  • 用途別トークン: 例: name, email, username, current-password など
checked
意味: 初期チェック状態にします。
指定方法: 属性名だけ書く（例: checked）
選択肢:
  • 付与: 有効
  • 省略: 無効
colorspace
意味: 用途に応じた値を指定します。
指定方法: colorspace="値"
dirname
意味: 任意のテキストを指定します。
指定方法: dirname="値"
disabled
意味: 操作できない状態にします。
指定方法: 属性名だけ書く（例: disabled）
選択肢:
  • 付与: 有効
  • 省略: 無効
form
意味: 用途に応じた値を指定します。
指定方法: form="値"
formaction
意味: URLを指定します。
指定方法: formaction="値"
formenctype
意味: URLを指定します。
指定方法: formenctype="値"
formmethod
意味: 用途に応じた値を指定します。
指定方法: formmethod="値"
formnovalidate
意味: 指定の有無で有効/無効を切り替えます。
指定方法: 属性名だけ書く（例: formnovalidate）
選択肢:
  • 付与: 有効
  • 省略: 無効
formtarget
意味: 用途に応じた値を指定します。
指定方法: formtarget="値"
height
意味: 表示高さを指定します。
指定方法: height="値"
list
意味: 用途に応じた値を指定します。
指定方法: list="値"
max
意味: 最大値を指定します。
指定方法: max="値"
maxlength
意味: 数値を指定します。
指定方法: maxlength="値"
min
意味: 最小値を指定します。
指定方法: min="値"
minlength
意味: 数値を指定します。
指定方法: minlength="値"
multiple
意味: 複数選択や複数入力を許可します。
指定方法: 属性名だけ書く（例: multiple）
選択肢:
  • 付与: 有効
  • 省略: 無効
name
意味: 要素の名前（送信時のキーなど）を指定します。
指定方法: name="値"
pattern
意味: 入力パターンを正規表現で指定します。
指定方法: pattern="値"
placeholder
意味: 入力欄のヒントを表示します。
指定方法: placeholder="値"
popovertarget
意味: 用途に応じた値を指定します。
指定方法: popovertarget="値"
popovertargetaction
意味: 用途に応じた値を指定します。
指定方法: popovertargetaction="値"
readonly
意味: 読み取り専用にします。
指定方法: 属性名だけ書く（例: readonly）
選択肢:
  • 付与: 有効
  • 省略: 無効
required
意味: 必須入力にします。
指定方法: 属性名だけ書く（例: required）
選択肢:
  • 付与: 有効
  • 省略: 無効
size
意味: 数値を指定します。
指定方法: size="値"
src
意味: リソースのURLを指定します。
指定方法: src="値"
step
意味: 増減単位を指定します。
指定方法: step="値"
type
意味: 要素の種類を指定します。
指定方法: type="値"
value
意味: 送信される値や初期値を指定します。
指定方法: value="値"
width
意味: 表示幅を指定します。
指定方法: width="値"

コード例

<label>名前 <input type="text" name="name"></label>
