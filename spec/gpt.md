# 仕様（GPT向け）

## サイト概要
- HTMLタグの解説を目的とした学習用サイト

## 必須要件
- `index.html` に各タグのリストと解説ページへのリンクを掲載する
- `tag/` 配下に `<各タグ>.html` を作成する
- 各タグページに以下を含める
  - タグの説明
  - 使い方がわかるコード例（コードブロック）
- 対象タグはHTMLの全タグとする
- タグ一覧の基準はWHATWG HTML Living StandardのIndices（Elements）
- Obsolete features に掲載される廃止・非推奨要素は対象外
- ここでのチャット内容は `spec/chat.md` に記録し、以後の会話も追記する
- 各タグページに属性一覧（意味・指定方法・選択肢）をコード例の前に記載する
- 固有属性は折りたたみ表示にする
- 共通属性一覧は `common-attributes.html` に配置し、`index.html` から参照できるようにする
- 選択肢がある属性は、各値の意味も説明する
- 外部ファイルを参照できるタグには、外部ファイルの読み込み方法も記載する
- 以後の仕様変更があれば `spec/plan.md` / `spec/gpt.md` / `README.md` を自動更新する
- 固有属性の説明は初心者向け（教科書レベル）に詳述する
- 固有属性の「何を表すか」をより詳細に記載する
- 途中の説明やデバッグメッセージは英語ではなく日本語で記載する
- 処理中に表示される思考メッセージ（例: assessing... など）は日本語で表示する

## ディレクトリ構成（予定）
- `index.html`
- `common-attributes.html`
- `style.css`
- `tag/`
  - `<各タグ>.html`
- `spec/plan.md`
- `spec/gpt.md`
- `spec/chat.md`

## 表記
- 日本語で説明する
- コード例はHTMLのコードブロックで記載する
