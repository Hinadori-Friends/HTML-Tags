# 計画

## 目的
HTMLタグの解説サイトを作成する。

## 方針
- `index.html` に各タグの一覧と解説ページへのリンクを掲載する
- `tag/` 配下に `<各タグ>.html` を作成し、説明とコード例（コードブロック）を載せる
- 対象タグはHTMLの全タグとする
- タグ一覧の基準はWHATWG HTML Living StandardのIndices（Elements）とする
- 廃止・非推奨（Obsolete features）の要素は対象外とする

## 進め方
1. 対象タグ（全タグ）の抽出とページ構成の決定
2. `index.html` の雛形作成
3. `tag/` 配下の個別ページ作成
4. 体裁の調整（見出し、ナビゲーション、コード例の整備）
