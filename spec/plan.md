# 計画

## 目的
- HTMLタグの解説サイトを作成し、初心者が理解できる説明とコード例を提供する

## 要件整理
- `index.html` にタグ一覧と各タグページへのリンクを掲載する
- `tag/` 配下に `<各タグ>.html` を作成する
- 各タグページに、説明・属性一覧（意味/指定方法/選択肢）・コード例を載せる
- 固有属性は折りたたみ表示にする
- 共通属性は `common-attributes.html` にまとめ、`index.html` から参照する
- 外部ファイルを参照できるタグは、その読み込み方法も記載する
- 固有属性の説明は初心者向けに具体的に記述する
- 以後の仕様変更があれば `spec/plan.md` / `spec/gpt.md` / `README.md` を自動更新する
- チャットログは `spec/chat.md` に追記する

## 進行状況
1. 仕様の確認とファイル構成の整理（完了）
2. ページ構造・共通スタイルの整備（完了）
3. 各タグページの説明・属性詳細の拡充（進行中）
4. 校正と表示確認（進行中）
